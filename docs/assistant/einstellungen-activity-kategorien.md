# Activity-Kategorien

Hier definierst du die Top-Level-Kategorien für den Typ-Picker der Aktivitäten.

## Felder
- **Key**: Technischer Schlüssel der Kategorie (z. B. `travel-time`).
- **Label**: Anzeigename der Kategorie (z. B. „Reisezeit“).
- **Icon**: Material-Icon-Name für die Anzeige (optional).
- **Order**: Sortierung – kleinere Werte erscheinen weiter oben.
- **Beschreibung**: Freitext für interne Hinweise.

## Hinweise
- Änderungen wirken direkt auf den Typ-Picker in der Planung.
- „Werkseinstellungen“ lädt die Kategorien aus den YAML-Defaults.
