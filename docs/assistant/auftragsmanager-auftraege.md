# Auftragsmanager · Aufträge

Kurze Hilfe zur Auftragsübersicht. Die Abschnitte sind kompakt gehalten, damit der Assistant sie gezielt verwenden kann.

## Wo finde ich das?

Navigation:

- **Auftragsmanager → Aufträge**
- Startseite `/` führt direkt in die Auftragsübersicht.

## Überblick

- Oben: Hero-Kennzahlen (Aufträge, Positionen, Hinweise).
- Mitte: Filterleiste (Suche, Status, Verantwortliche, Sortierung).
- Unten: Auftragskarten mit Tabs (Details, Positionen, Dokumente).
- Button **Neuer Auftrag** öffnet den Erstellen-Dialog.

![Auftragsübersicht](/assistant/auftragsmanager/auftraege-uebersicht.png)
Die Übersicht zeigt oben die KPIs, darunter Filter und die Auftragskarten.

## Neuer Auftrag

Schritt-für-Schritt:

1. **Auftragsmanager → Aufträge** öffnen.
2. Button **Neuer Auftrag** klicken.
3. Im Dialog **Name** ausfüllen (Pflichtfeld).
4. Optional **Auftrags-ID** setzen (leer = automatische ID).
5. **Fahrplanjahr** auswählen.
6. Optional **Kunde**, **Tags**, **Kommentar** pflegen.
7. Mit **Anlegen** speichern.

![Dialog Neuer Auftrag](/assistant/auftragsmanager/auftrag-neu-dialog.png)
Der Dialog enthält Pflichtfeld **Name** sowie optionale Felder wie Auftrags-ID, Kunde, Tags.

## Auftrag in der Liste prüfen

1. Nach dem Speichern erscheint der neue Auftrag als Karte in der Übersicht.
2. Mit der Suche oder Tags filtern, falls viele Aufträge vorhanden sind.

## Positionen (Auftragskarte)

1. Eine Auftragskarte anklicken, um sie aufzuklappen.
2. Tab **Positionen** öffnen.
3. Über **Auftragsposition hinzufügen** neue Positionen anlegen oder zuordnen.

![Auftragskarte mit Positionen](/assistant/auftragsmanager/auftrag-karte-positionen.png)
In der Karte siehst du den Button **Position** und die darunterliegenden Positionen.

## Auftragsposition hinzufügen

Schritt-für-Schritt:

1. In der Auftragskarte **Position** klicken.
2. Dialog öffnet sich mit Tabs (Leistung, Fahrplan, Fahrplan manuell, Import).
3. Pflichtfelder je Tab ausfüllen (mit * markiert).
4. **Speichern** klicken.

![Dialog Auftragsposition hinzufügen](/assistant/auftragsmanager/auftrag-position-dialog.png)
Der Dialog bietet Tabs (Leistung, Fahrplan, Fahrplan manuell, Import) mit Pflichtfeldern.
